Resumo

Uma API, ou Interface de Programação de Aplicativos, é o cerne deste estudo, que tem como objetivo o desenvolvimento de uma API para classificação automatizada da Retinopatia Diabética. Esta condição oftalmológica grave, se não diagnosticada precocemente, pode levar à perda de visão. A API é projetada para integração com sistemas de saúde e diagnóstico, simplificando o processo de diagnóstico da doença. Ela utiliza uma Rede Neural Profunda em conjunto com modelos de pré-processamento estabelecidos. A metodologia engloba a criação de um sistema de segurança sólido e documentação completa para proteger dados sensíveis dos pacientes e facilitar a compreensão e utilização da API. Baseado em pesquisas relacionadas, este estudo almeja contribuir para diagnósticos mais precisos e detecção precoce da Retinopatia Diabética.
